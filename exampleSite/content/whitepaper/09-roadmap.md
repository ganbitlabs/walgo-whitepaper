## Phase 1 — Foundation (Q3-Q4 2024)

**Status:** Completed

- [x] Core CLI framework with Cobra and Viper
- [x] Hugo integration — init, build, serve, new content
- [x] HTTP deployment mode (no wallet required)
- [x] On-chain deployment via site-builder
- [x] Brotli compression pipeline
- [x] HTML, CSS, and JavaScript optimization
- [x] Basic project tracking
- [x] Cross-platform installation scripts (macOS, Linux, Windows)

## Phase 2 — AI & Themes (Q1-Q2 2025)

**Status:** Completed

- [x] AI content generation with OpenAI integration
- [x] AI pipeline — plan, generate, resume, fix
- [x] OpenRouter provider support
- [x] Biolink theme — link-in-bio pages with TOML configuration
- [x] Obsidian vault import with wikilink conversion
- [x] Interactive launch wizard (8-step guided deployment)
- [x] SuiNS domain management
- [x] Walrus RFP selection and official support

## Phase 3 — Desktop & Management (Q3-Q4 2025)

**Status:** Completed

- [x] Desktop application with Wails (Go + React/TypeScript)
- [x] Full project management — list, show, edit, archive, delete
- [x] Project update and redeployment by name or ID
- [x] Doctor command for environment diagnostics
- [x] Dependency auto-installation (Hugo, Sui, site-builder, walrus)
- [x] Setup-deps command for tool management
- [x] Status command for deployment verification
- [x] Version checker with GitHub releases integration
